# Workflow: UI Variations
Generate 4-5 unique design variations for a component.

**Variations to produce:**
1. **Minimalist Glass**: Frosted glass, subtle transparency.
2. **Elevated Corporate**: Strong shadows, gold/amber accents.
3. **Modern Compact**: Space-efficient, mobile-first.
4. **Outlined**: Border-centric, clean line work.
5. **Colorful Gradient**: Vibrant, high-energy.

Maintain the exact same component interface and functionality for all variations.
